# Rocket 15 Questions

An unofficial H5P content type for gamified practice in the classroom.

Learners choose a graphic at the start (rocket, butterfly, owl, or ice cream cone). Each correct answer reveals another part of the graphic. After 15 correct answers the graphic is complete and a closing animation plays.

## Question types

- **Single Choice** – one correct answer from multiple options
- **True / False** – evaluate a statement as true or false
- **Short Answer** – learners type a short text answer (case-insensitive)
